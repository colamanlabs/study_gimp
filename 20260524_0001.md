


### 03
##### 김프 이미지열기에서 파일목록 형태가 모두 자세히 형태 입니다. 라인형태로 위에서 아래로 1개씩 나오는 것이요. 이것을 아이콘 형태로 바꿀 수 있습니까?
```aiignore
GIMP 3.x의 파일 열기 창은 Windows 기본 탐색기와 조금 달라서, GIMP 자체 설정인지 GTK 파일 선택창 기능인지 나눠서 확인해 보겠습니다.
가능은 제한적입니다.
GIMP의 파일 > 열기 창은 Windows 탐색기가 아니라 GTK 파일 선택창이라서, Windows 탐색기처럼 “큰 아이콘 / 중간 아이콘 / 자세히”를 자유롭게 바꾸는 기능이 보통 없습니다. GIMP 쪽 설정에서도 파일 목록을 아이콘형 썸네일 목록으로 바꾸는 옵션은 뚜렷하지 않습니다. GTK 파일 선택창 자체도 오랫동안 아이콘/썸네일 보기 지원이 부족했던 것으로 알려져 있습니다.
대신 현실적인 방법은 이것입니다.
Windows 탐색기에서 이미지 폴더를 아이콘 보기로 열기
원하는 이미지를 선택
GIMP로 드래그 앤 드롭하거나, 우클릭 → 연결 프로그램 > GIMP
GIMP 안에서는 오른쪽 미리보기 패널을 보고 선택
즉, GIMP 파일 열기 창 자체를 탐색기처럼 아이콘 격자형으로 바꾸기는 어렵고, 탐색기에서 고른 뒤 GIMP로 여는 방식이 가장 편합니다.
```


#### 03-1 이미지 회전하기
```aiignore
메뉴바 > 이미지 > 변형 > 회전(시계방향, 시계반대방향)

단축키
ctrl + 마우스 휠 -> 확대/축소
```

#### 03-3 이미지 사이즈 변경
```aiignore
포토샵 이미지 사이즈와 같은기능

메뉴바 > 이미지 > 이미지 크기조정
```

#### 03-4 이미지 압축 
```aiignore
JPEG 형식으로 내보내면 이미지 압축률을 선택할 수 있다.

좌측상단에 옵션 슬라이더 사용한다.
```


#### 03-5 이미지 자르기
```aiignore
포토샵 크롭툴 과 같다.

도구상자 크롭툴 선택하고, 
마우스 드래그로 영역지정
모서리를 선택하면 크롭 된다.
```

#### 03-6 이미지 뒤집기
```aiignore
도구상지 변형툴의 확장메뉴에 있다. 
```
