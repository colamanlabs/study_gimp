#### 01-2 김프 환경 설정 
```
1. 테마선택
편집 > 기본설정 > 인터페이스 > 테마 > 우측 테마 패널 
- Default 는 다크계열
- Sytstem 선택한다.


2. 아이콘 크기 설정 
편집 > 기본설정 > 인터페이스 > 테마 > 우측 테마 패널
가운데 "Override icon sizes set by the theme" 체크박스 활성화 하고 
아래 슬라이더 변경 한다.
기본값이 Medium 인데 적당하다.


3. 도구상자 1
편집 > 기본설정 > 인터페이스 > 도구상자 > 우측 도구상자 패널 
체크박스 활성화 한다
1) "활성화된 붓, 무늬, 그라디언트 표시" -> 체크로 활성화  
2) "활성화된 이미지 표시 " -> 체크로 활성화

좌측 상단 도구상자(툴박스) 아래 전경색/배경색 아이콘 옆에 아이콘 2개 추가된 것 확인한다.


4. 도구상자 2
편집 > 기본설정 > 인터페이스 > 도구상자 > 우측 도구상자 패널

가운데 도구설정 목록에서 선택한다. (선택하면 좌측에 눈 아이콘으로 checked 됨을 알 수 있다.)
1) 명도-대비
2) 임계값
3) 레벨
4) 커브 

선택해도 도구상자에 나타난 변경사항은 없다.
```


#### 01-3 김프 워크플로우

```aiignore
이미지 파일 로딩할 때 컬러 프로파일 변경창 뜨면 변경(Convert) 선택한다.

ctrl + z --> undo
ctrl + y --> redo


프로젝트 내보내기와 웹 게시파일은 다르다.

다른이름으로 저장하기 -> 프로젝트 파일를 저장 
내보내기 -> png 로 파일 생성

작업파일이면 xcf 로 내보낸다.
포토샵의 psd 파일과 역할이 같다.
```

```
GIMP 의 기본 저장 포맷은 XCF(eXperimental Computing Facility) 입니다.
확장자는 .xcf 를 사용합니다.
예:
image.xcf
banner.xcf
XCF 는 GIMP 전용 작업 파일 형식으로, 작업 중간 상태를 그대로 저장하기 위한 목적입니다.

gimp 확장자명을 eXperimental Computing Facility 의 약자로 한 배경이 있습니까?

네. 흥미로운 배경이 있습니다.
사실 XCF 는 원래 이미지 포맷 이름으로 만든 것이 아니라, 초기 개발 환경 이름에서 가져온 것입니다.

GIMP 는 1995년에 미국 University of California, Berkeley 학생들이 만든 프로그램인데, 당시 개발 환경이었던 XCF(X eXperimental Computing Facility) 에서 이름을 가져왔습니다.

즉:

XCF = 원래 버클리의 컴퓨팅 연구 시설 이름
GIMP 개발이 그 환경에서 이루어짐
그래서 자체 파일 포맷 이름도 XCF 로 부름

입니다.
```
